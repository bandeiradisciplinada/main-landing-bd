# Bandeira Disciplinada

Welcome to the official repository for the **Bandeira Disciplinada** corporate website and product portfolio. 

At BD, we believe that technology can be a powerful tool for improving people's lives. Our team is dedicated to developing cutting-edge AI software solutions that solve real-world problems.

## 🚀 Our Products
This website serves as the central hub for our diverse ecosystem of products:
- **JobJa**: AI-powered recruitment and career advancement platform.
- **ZeroG**: Mobile educational application using spaced repetition algorithms.
- **SubFlow**: Professional-grade, privacy-centric local AI subtitle studio.
- **Portugal Tools Hub**: Utility platform for expats and locals in Portugal.
- **Namak**: All-in-one smart event management platform.

## 🛠 Tech Stack
This project is built using modern web development standards:
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) with a custom glassmorphism aesthetic
- **Language:** TypeScript
- **Package Manager:** Yarn

## 📦 Getting Started

First, install the necessary dependencies:

```bash
yarn install
```

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3002](http://localhost:3002) (or the port specified in your terminal) with your browser to see the result.

## 🗂 Project Structure
- `src/pages/`: Contains all Next.js routes, including dynamic routing for our products (`/products/[slug].tsx`).
- `src/components/`: Reusable UI components (Headers, Footers, Product Cards, etc.).
- `src/config/`: Contains `index.json`, which acts as the central data source (CMS) for the entire website's content, links, and product details.
- `public/`: Static assets, images, and optimized product gallery screenshots.
